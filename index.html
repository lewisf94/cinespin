🎬 CineWheel

⚙️ One quick setup step

CineWheel needs a free Firebase project to store your groups' data. It takes about 5–10 minutes, once.

  1. Go to console.firebase.google.com and create a project.
  2. Add a Web app (the </> icon) and copy the firebaseConfig values.
  3. Open Build → Firestore Database and create a database.
  4. Paste the contents of firestore.rules into the Firestore Rules tab and publish.
  5. Open Build → Authentication → Sign-in method and enable Anonymous.
  6. Paste your config into js/firebase.js and reload.

Full click-by-click instructions are in README.md.
